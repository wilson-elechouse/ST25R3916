# ST25R3916

This repository keeps the original archive and also provides the extracted source files for easier browsing and use.

## Contents

- `ESP32_ST25R3916_sample code.zip`
  - Original archive uploaded to this repository.
- `ESP32_ST25R3916_reading_14443AB_15693/`
  - ESP32 sample sketch for reading ISO14443A, ISO14443B, and ISO15693 tags.
- `NFC-RFAL/`
  - RFAL library source used for ST NFC reader integration.
- `ST25R3916_110/`
  - ST25R3916/ST25R3916B library source.
- `testingCode_LED/`
  - Simple LED testing example.

## Notes

- The original zip file is intentionally kept unchanged.
- The extracted folders are added so the code can be viewed directly on GitHub and used without manual extraction.
